# 🚀 Playwright Test Automation Framework

This project is a modern UI test automation framework built with **Playwright**, **Python**, and **Pytest**. It demonstrates best practices for designing scalable, maintainable, and reliable automated UI tests using the **Page Object Model (POM)**.

The framework automates key user workflows in the SauceDemo application, including login, inventory management, shopping cart validation, and the complete checkout process. It was developed to explore Playwright's capabilities and compare its architecture and developer experience with a Selenium-based automation framework.
